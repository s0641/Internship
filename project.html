Design and Development of Smart City Solution for townships and Smart Cities

THE SCOPE

1. Multi Level project Monitoring and Tracking system

2. Interactive user friendly web application software that will allow to Keep all hierarchy in the organization, namely Higher Management ,Mid Level Manager, Project leaders, Project executors and all other concerned Employees

3. Project Creation and administration

4. Creation of Different stages of Project and updates

5. Assigning Stakeholders of the Project

6. Creation of Milestones and sub-milestones up to Nth Level

7. Powerful messaging Module to inform every assigned resources through various means of communication like Email and SMS

8. Creation of Active directory and assigning of resources to each level.

THE CHALLENGE

1. Top Management should have 24/7 real-time access to various projects and various view should be facilitate addressing any issues or bottleneck which needs attention.

2. App for anytime access to information and to update tasks on real-time.

3. Automatic Messing system to inform respective stake holders or the project for events and updates.

4. Nth Level hierarchy for Users and Tasks which can be further assigned to various levels.

5. Powerful dashboard for each type of user to cater to the needs of the requirement of the user.

6. Reporting Module to access various on demand and ad-hoc reports.

7. Forecasting/Alerting Mechanism in the application to forecast and alert the progress or any Warning which is required by the stake holders of the application.
