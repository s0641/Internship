JVVNL – Jaipur Vidyut Vitran Nigam Limited, Jaipur, Rajasthan

THE SCOPE

1. Download CMRI Data to Base Computers

2. Centralized single point parsing of downloaded files of various meter makes

3. Convert the data to Common Data Format

4. Create a RDBMS with all the data stored centrally

5. Archive data files and submit

6. Single Web-Application to view all the meter data in standard formats

7. Wiring Verification, Load Survey Analysis and Load Utilization Pattern Anaylsis

8. MIS reports

THE CHALLENGE

1. Managing the CMRI’s and CMRI Data

2. Building assessment and VEE Rule Engine.

3. Understanding MIOS CDF format and

4. Manpower to Complete All Reading within first few days of every month

THE SOLUTION

1. A Web based enterprise Grade software was built to cater to all the requirements.

2. A fully functional fully compliant MOIS standard CDF parser was built.

3. CMRI Manager, Seal Manager, Data Dumping tools with extensive reporting was built .

4. Dashboard was developed which gives to bird view for the Meter Data analysis.

5. Load Survey Comparison with Billing Parameters
6. Usage Index :- Load Factor, Average Demand, Max Demand

7. Blackout & No Load Demand patterns

8. Analysis on Tamper Events, Meter Faults, Phase association errors, Seal Open and other events, Load Violations and Missing interval Data Assessment.

THE HIGHLIGHTS

1. A Fully web based MDA software was developed and deployed.

2. More then 12,000+ Meters with different make such as L&T, Secure, Genus, HPL, L&G etc with full meter data analytics.

3. All High Valued customers such as HT, MIP and SIP are covered

4. Entire Solution is executed in strict timelines HT 3 Days, MIP-5 Days and SIP 7 Days..

5. The entire software developed using Open source technologies

6. Quality Checking of data was done with advance web based tools.

7. Validation, Estimation and Edition(VEE).
